pygame.math
    pygame module for vector classes
      pygame.math.Vector2 - a 2-Dimensional Vectora 2-Dimensional Vector
      pygame.math.Vector3 - a 3-Dimensional Vectora 3-Dimensional Vector

    The pygame math module currently provides Vector classes in two and three dimensions, Vector2 and Vector3 respetively.

    New in Pygame 1.10
